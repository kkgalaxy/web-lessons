Розмітка заголовків

Заголовки виконують важливу функцію на веб-сторінці. По-перше, вони показують важливість розділу, до якого належать. Чим більше заголовок та його «вага», тим більше він значимий.

Найвищим рівнем є рівень 1 ( <h1> ), а найнижчим - рівень 6 ( <h6> )). - заголовки нижчіх рівнів, розташовані за спаданням їх важливості.

Синтаксис:

<h1> Заголовок 1-го рівня </h1>

<h2> Заголовок 2-го рівня </h2>

<h3> Заголовок 3-го рівня </h3>

<h4> Заголовок 4-го рівня </h4>

<h5> Заголовок 5-го рівня </h5>

<h6> Заголовок 6-го рівня </h6>

Результат

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня


Форматування тексту

<strong> — HTML використовується для виділення тексту жирним шрифтом. У HTML тег слід використовувати виділення важливих фрагментів тексту за змістом (логічного виділення).

Синтаксис:

<strong> важный фрагмент текста </strong>

Результат:

важный фрагмент текста

<del> — HTML вказує фрагмент тексту, який був видалений з нової версії документа (використовувався в старій версії). Причина видалення може бути у документі зазначеному в атрибутіцитувати тега, а дата/час видалення фрагмента - в атрибуті дата, час.

Синтаксис:

<del>12.03.2013<del> 13.03.2013

Результат:

12.03.2013 13.03.2013

<small> — HTML використовується для написання тексту шрифтом меншого розміру.При використанні HTML тега розмір шрифту зменшується щодо розміру шрифту батьківського тега (розміру тексту навколо тега малий).

Синтаксис:

<small>*Только для сотрудников</small>

Результат:

*Только для сотрудников

<sub> — HTML використовується для написання тексту у вигляді нижнього індексу.

Синтаксис:

Химическая формула воды - это H<sub>2</sub>O, а молекулярного кислорода - O<sub>2</sub>.

Результат:

Химическая формула воды - это H2O, а молекулярного кислорода - O2.

<sup> — HTML використовується для написання тексту у вигляді верхнього індексу.

Синтаксис:

Теорема Пифагора для прямоугольного треугольника: a<sup>2<sup>+ b<sup>2<sup> = c<sup>2<sup>.

Результат:

Теорема Пифагора для прямоугольного треугольника: a2 + b2 = c2.

<ins> — HTML вказує фрагмент тексту, який був доданий у новій версії документа (раніше його не було). Причина додавання/зміни фрагмента може бути у документі зазначеному в атрибутіцитувати тега , а дата/час додавання - в атрибуті дата, час.

Синтаксис:

<ins>12.02.2013</ins>

Результат:

12.02.2013


Введення "комп'ютерного" тексту

<kbd> — HTML використовується для позначення тексту як клавіатурного введення. Браузери зазвичай відображають текст усередині HTML тега моноширинний шрифт.

Синтаксис:

Комбинация клавиш <kbd><Ctrl> + <V></kbd>позволяет вставить данные из буфера обмена.

Результат:

Комбинация клавиш <Ctrl> + <V> позволяет вставить данные из буфера обмена.

<em> — HTML використовується виділення тексту курсивом - похилим шрифтом. У HTML тег слід використовувати для акцентування уваги, логічного "підкреслення" фрагментів тексту

Синтаксис:

<em>виділений фрагмет тексту</em>

Результат:

виділений фрагмет тексту

<samp> — HTML визначає текст як результат виведення комп'ютерної програми.

Синтаксис:

В ходе обработки запроса были получены следующие результаты:
Обработано объектов: 10582,
Найдено подходящих объектов: 28.

Результат:

В ходе обработки запроса были получены следующие результаты:
<samp>Обработано объектов: 10582,
Найдено подходящих объектов: 28</samp>.

<pre> — HTML використовується для виведення тексту зі збереженням формату, в якому текст був написаний у текстовому редакторі.

Синтаксис:

<pre>Текст внутри тега pre выводится с  тем     количеством    пробелов и с теми позициями строк, что были заданы в редакторе.</pre>

Результат:

Текст внутри тега pre выводится с  тем     количеством    пробелов и с теми позициями строк, что были заданы в редакторе.

<code> — HTML використовується для позначення тексту як програмного коду.

Синтаксис:

<code>      <style type="text/css">
        .main {
          width: 100%;
          background-color: white;
          font-size: 14px;
        }
        </style></code>

Результат:

    
        <style type="text/css">
        .main {
          width: 100%;
          background-color: white;
          font-size: 14px;
        }
        </style>


Оформлення цитат та визначень

<dfn> — HTML вказує, що текст усередині є терміном.

Синтаксис:

<dfn title="Hypertext Markup Language">HTML></dfn> - это язык разметки веб-страниц.

Результат:

HTML - это язык разметки веб-страниц.

<q> — HTML вказує, що текст всередині є цитатою.

Синтаксис:

<q>Выбери себе работу по душе, и тебе не придется работать ни одного дня в своей жизни</q> - Конфуций.

Результат:

Выбери себе работу по душе, и тебе не придется работать ни одного дня в своей жизни - Конфуций.


<hr> — використовується в HTML для виведення горизонтальної лінії – роздільника.

Синтаксис:

<p>Некий текст на странице.<p>

<hr>

<p>Текст, который по смыслу отделен от текста выше.<p>

Результат:

Некий текст на странице.

Текст, который по смыслу отделен от текста выше.

<abbr> — використовується в HTML коді для визначення вмісту тега як абревіатури або акроніму (вказує, що текст у тегу – це абревіатура). В атрибутітитул вказують розшифрування.

Синтаксис:

<abbr> title="World Wide Web">WWW><abbr>

Результат:

WWW

<blockquote> — використовується як обгортка тексту, що цитується, вказує, що текст всередині є цитатою. Атрибутцитувати вказує на джерело цитування.

Синтаксис:

<blockquote cite="http://example.com/citata/">Это текст, процитированный с другого сайта</blockquote>

Результат:

Это текст, процитированный с другого сайта

<br> — вказує місце перенесення рядка у тексті HTML сторінки. Тег br ставиться перед місцем кінця рядка, текст після тега буде виведено з нового рядка.

Синтаксис:

Это первая строка обычного текста на веб-странице. В конце перенос строки<br> И вот уже вторая строка текста.

Результат:

Это первая строка обычного текста на веб-странице. В конце перенос строки
И вот уже вторая строка текста.

<cite> — визначає вміст як назву іншого матеріалу. За допомогою HTML тегу <cite> можна виділяти назви пісень, фільмів, книг, телепередач, веб-сайтів, картин тощо.

Синтаксис:

В 2009-м году Джеймс Кэмерон снял всемирно известный фильм - Аватар.

Результат:

В 2009-м году Джеймс Кэмерон снял всемирно известный фильм - Аватар.


Cписки

<li> — HTML визначає елемент (пункт) списку.

<ul> — HTML визначає маркований список (неупорядкований список).

Синтаксис:

        <ul>
            <li>Мочалка</li>
            <li>Іграшка</li>
            <li>Молоток</li>
            <li>Мочалка</li>
            <li>Іграшка</li>
            <li>Молоток</li>
            <li>Мочалка</li>
            <li>Іграшка</li>
        </ul>

Результат:

  • Меркурий
  • Венера
  • Земля
  • Марс
  • Юпитер
  • Сатурн
  • Уран
  • Нептун

<ol> — HTML визначає нумерований список.

Синтаксис:

        <ol>
            <li>Создание дизайна</li>
            <li>Верстка</li>
            <li>Интеграция</li>
            <li>Продвижение</li>
        </ol>

Результат:

  1. Создание дизайна
  2. Верстка
  3. Интеграция
  4. Продвижение


Силки

<a> — використовується для визначення посилання (гіперпосилання, якоря) у HTML документі. За допомогою тега<a> можна посилатися на інші сторінки сайту та/або на конкретний елемент на сторінці (посилання – якір).

Синтаксис:

<a href="https://guruweba.com/html">Справка HTML</a>

Результат:

Справка HTML

Спеціальні символи

&amp; — символ амперсанта

Синтаксис:

Параграф &223

Результат:

Параграф &223

&copy; — символ copyright

Синтаксис:

&copy; Красаков К.А., 2021

Результат:

©Красаков К.А., 2021

&lt; — знак менше

Синтаксис:

4532 &lt; 5000

Результат:

4532 < 5000

&trade; — символ торгової марки

Синтаксис:

RapidTables &trade;

Результат:

RapidTables ™

&gt; — символ більше

Синтаксис:

200 &gt; 150

Результат:

200 > 150

&nbsp; — нерозривний пробіл

Синтаксис:

Кирило&nbsp;Красаков

Результат:

Кирило Красаков
